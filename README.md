# 🩺 Diabetes Assistant Chatbot

A smart AI-powered Diabetes Prediction Chatbot built with Python and Streamlit.

## 🌟 Features
- Conversational chatbot interface
- Bilingual support (English & Hindi)
- Real-time diabetes prediction using ML model
- Health tips and doctor recommendations
- Progress tracker in sidebar
- Trained on Kaggle Diabetes Dataset

## 🛠️ Tech Stack
- Python
- Scikit-learn (Machine Learning)
- Streamlit (Web App)
- Pandas & NumPy

## 📊 ML Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting ✅ (Best Accuracy: 75.32%)
- SVM, KNN, Decision Tree

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run Our_chatbot.py
```
